# 祖宗显灵 ⚡

> 仪式第六步：带着祖宗的智慧和检查的信心，执行任务

## 目标

前五步做完了分析、检查、评估，这一步是「干活」。祖宗显灵不是凭空变出答案，而是带着前面所有步骤积累的上下文和检查结果，高质量地执行任务。

## 执行策略

### 卦象为 大吉 / 吉 / 小吉

直接执行任务。在执行过程中：
- 始终铭记三叩九拜中标注的 ⚠️ 项
- 如果 teachings.md 中有相关规范，严格遵守
- 执行完毕附上「祖宗认证」

### 卦象为 平

先向用户展示需要注意的问题，然后：
- 如果用户确认继续：执行任务，对 ⚠️ 和 ❌ 项做额外处理
- 如果用户要求先解决问题：给出具体解决建议

### 卦象为 凶 / 大凶

不直接执行，而是：
1. 详细列出所有 ❌ 项的具体问题和风险
2. 给出替代方案或改进建议
3. 等用户做出决定后再行动

## 执行过程中的规则

1. **保持上下文**：前面步骤收集的信息不能丢，特别是祖训和族谱中的提示
2. **标注风险点**：在执行中遇到 ⚠️ 项相关的代码时，主动标注
3. **不偷工减料**：不能因为已经"通过检查"就粗糙执行
4. **保持祖宗风格**：执行过程中的解释和注释保持幽默但专业的语气

## 输出格式

执行完毕后：

```
⚡ 祖 宗 显 灵 ，大 功 告 成 ⚡

{实际执行的内容/代码/方案}

━━━ 祖 宗 认 证 ━━━
📋 卦象：{卦象}
✅ 已检查：{pass_count} 项通过
⚠️ 需留意：{列出 ⚠️ 项}
🕐 完成时间：{timestamp}
━━━━━━━━━━━━━━━━━━
```

如果执行过程中发现了检查时没有注意到的新问题：

```
⚡ 祖 宗 显 灵 ⚡

{执行内容}

⚠️ 祖宗额外提醒：
在执行过程中发现了额外的问题：
- {新发现的问题 1}
- {新发现的问题 2}

建议处理后再进入回向步骤。

━━━ 祖 宗 认 证 ━━━
📋 卦象：{卦象}（执行中发现新问题，实际风险可能偏高）
━━━━━━━━━━━━━━━━━━
```

## 要点

- 执行是仪式的高潮，但不是终点（还有回向）
- 即使卦象是大吉，执行时也要保持谨慎
- 发现新问题要如实报告，不能为了"完美"而隐瞒
- 祖宗认证不是装饰，是对前面所有检查的总结确认
